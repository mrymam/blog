# Blog

Astroを使用した個人ブログサイトです。GitHub Pagesでホスティングしています。

## 概要

- **フレームワーク**: [Astro](https://astro.build/)
- **ホスティング**: GitHub Pages
- **デプロイ**: GitHub Actions

## 開発環境

開発環境のセットアップについては [dev/README.md](dev/README.md) を参照してください。

## AIと自動化の活用

- Claude Codeを使用した開発支援
- GitHub Actionsによる自動デプロイ

## セキュリティ

- 機密情報は環境変数で管理
- `.env`ファイルはコミットしない
- 依存パッケージは定期的に更新
